## ✨ Sobre o projeto

**Lyra** é uma aplicação web desenvolvida para ajudar músicos a organizarem suas apresentações de forma simples, intuitiva e eficiente.

Inspirado na constelação **Lyra** — e no instrumento musical **lira** — o sistema busca trazer harmonia e estrutura ao fluxo musical.
